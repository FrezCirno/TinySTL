# 说明

此文件夹内为一个仿STL、简化版的 C++ 数据结构模板库，用在本次数据结构课程设计中

### 已实现的功能

##### 基本功能

- allocator
- construct
- uninitialized
- rbtree 
- heap
- pair
- type_traits


##### 数据结构模板

- vector
- list
- stack
- deque
- queue
- priority_queue
- set
- map


### 未实现的功能
